有一个 VSCode 的小插件，叫 [Markdown Padding](https://marketplace.visualstudio.com/items?itemName=harttle.md-padding-vscode)，
安装好后，按下 Cmd(Windows) + Shift + P, 输入 markdown padding，即可快速为 markdown 的中英文及数字间添加空格，非常方便。
